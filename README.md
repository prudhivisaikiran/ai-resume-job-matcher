# AI Resume & Job Matcher

A full-stack MERN application for analyzing resumes and matching them with job descriptions using AI.

## Tech Stack
- **Backend:** Node.js, Express, MongoDB (planned)
- **Frontend:** React, Vite, Modern CSS
- **AI:** OpenAI API (planned)

## Roadmap

- [ ] Dashboard
- [ ] Deployment

## Quick Start

### Backend
```bash
cd backend
npm install
npm run dev
```
Runs on http://localhost:5000

### Frontend
```bash
cd frontend
npm install
npm run dev
```
Runs on http://localhost:5173

## Folder Structure
- `backend/`: API and server logic
- `frontend/`: React client
